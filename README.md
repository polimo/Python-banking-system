Python Banking System

A simple console-based banking system written in Python that allows users to create multiple accounts, generate passwords, deposit and withdraw money and view account details.

Features:

-Add multiple bank accounts
-Custom password generation
-Withdraw and Deposit
-Show accounts details (Without the password)
-Remove accounts

How to use:

1-Clone the repository or download the banking_system.py file.
2-Run the program with python3 ==> python banking_system.py

Notes:

This is a beginner friendly code, with potential bugs, Able to improve.
